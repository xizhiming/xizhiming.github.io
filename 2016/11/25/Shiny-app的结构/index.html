Shiny app的结构

发表于 2016-11-25   |   分类于 R

每个Shiny app包含两部分的文件：ui.R(a user-interface definition),server.R(a server script)。
ui.R定义app界面输入、输出格式和布局，server.R接受输入值，进行数据的处理计算，并将结果传递至ui.R进行展示。
